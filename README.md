## 🚀 部署 Deployment

本项目为纯静态 HTML 页面，可通过 **GitHub Pages** 一键部署：

This is a pure static HTML page that can be deployed with **GitHub Pages**:

1. 将仓库推送到 GitHub / Push this repo to GitHub
2. 进入仓库 Settings → Pages / Go to Settings → Pages
3. Source 选择 `main` 分支，保存 / Select `main` branch and save

## 📝 背景 Background

本页面为**"廉花郑放·赓续清风——红色廉洁文化档案展暨第十九个'6·9'国际档案日庆祝活动"**的配套学习资源导航页。

焦裕禄（1922–1964），山东淄博人，曾任河南省兰考县委书记。在兰考工作期间，他带领群众治理风沙、盐碱和洪涝灾害，积劳成疾，病逝于工作岗位，年仅42岁。他用实际行动诠释了"亲民爱民、艰苦奋斗、科学求实、迎难而上、无私奉献"的焦裕禄精神。
